# /go/ paths

## `/go/?keyboard=Keyboard_<id>&language=<name>`

The top level `/go/` path is used for redirects from KeymanWeb documentation.
Note that `/go?...` internally redirects to `/go/?...`, which is served by
index.php

## `/go/edit-page?page=url[&id=id][&file=file]`

Forwards requests to edit a page on the site to the appropriate edit page on
GitHub.
