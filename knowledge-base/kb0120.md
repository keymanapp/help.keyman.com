# HOWTO: Upgrade Keyman for Windows 18.0 (or earlier) to 19.0+ on ARM devices

## Summary

Keyman for Windows version 19.0.182 or later supports Windows 11 on ARM. If a previous version of Keyman is already installed that does not support native ARM applications (any version less than 19.0.182), it will first need to be uninstalled. Updating from non-supported versions to the latest will result in incorrect Windows registry settings.

## Previous Versions

“I thought I had a previous version of Keyman partially running on my ARM-based Windows device?”

Windows on ARM devices can run applications built for 32-bit (x86) and 64-bit (x64/amd64) architectures. These applications may work with older versions of Keyman, but this configuration is not officially supported.
